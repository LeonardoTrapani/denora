# Daenya

Bare Bun + Turborepo monorepo skeleton for Daenya.

## Structure

- `apps/*` for application surfaces when they are chosen.
- `packages/*` for shared packages when they are needed.
- `CONTEXT.md` captures the product direction.
- `AGENTS.md` captures repository guidance for agents.
- `opencode.json` configures the `effect-v4` reference repo.

No app framework is selected yet.

## Commands

```sh
bun install
bun fmt
bun lint
bun check-types
bun run build
```
